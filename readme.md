
<h1 align="center">hyprcursor-rs</h1>
<p align="center">a rust implementation of a not very good cursor format</p>

a rust implementation of the [hyprcursor format](https://standards.hyprland.org/hyprcursor/), a cursor format supporting svgs and pngs.

if you want to have a better cursor format with better backwards compatability, check out [kcursor](https://github.com/m4rch3n1ng/kcursor), a rust implementation of the kde svg cursor format.
